
1. To declared variables in javascript, var,let and const are used.

 var is a old way of declaring variables, which is not  block-scoped but can be redeclared and reassigned.

 let is block-scoped, can be reassigned but can not be redeclared in the same scope.

 const is used declaring constant variables. it is a block-scoped but can not be reassigned or redeclared.


2. The spread operator(..) is a syntax to expand the elements of an array or object, It is used to copy or marge an array or object.
It is also used to pass multiple values into a function.


3. map(), filter() and forEach() are array methods. where 
map() is used to create a new array.
filter() is used to create a new array with elements that pass given condition.
and forEach() is used to loop through array but does not create a new array.

4. An arrow function is a way to create a function which is shorter and more modern.

5. Template literals are a modern and efficient way to create strings where it makes complex strings much easy to handle and can work with variables, math or logic using ${}.
